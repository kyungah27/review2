# GIANTS BOX

### 1. 프로젝트명
    - Giants Box   

### 2. 목적
    - 수많은 소모임 플랫폼중에 ‘영화’에 초점을 맞춘 소모임 이벤트 사이트를 구축
    - 코로나가 종식되는 순간의 폭발적인 수요를 고려해 선점효과를 위한 개발 진행
    
### 3. 팀 구성
    - 팀장 : 최현우
    - 팀원 : 심예지, 송영진, 김도희, 곽경아, 전현정
    
#### 3.1. 담당업무
    - 1) 이벤트 전반 : 심예지
    - 2) 회원가입, 로그인 : 최현우
    - 3) 이미지 및 메인페이지 : 전현정
    - 4) 영화정보 및 이벤트참여 : 송영진
    - 5) 후기 : 곽경아
    - 6) 댓글 : 김도희
